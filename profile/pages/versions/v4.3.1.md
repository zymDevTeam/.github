# Release Notes - v4.3.1 🚀
_19 de marzo de 2026_

Esta versión incluye importantes mejoras en la administración de menús, la implementación de registros de auditoría detallados para POQTROL y la optimización de consultas en la base de datos de Mi Proceso.

---

## 🏗️ zymDev Core (General)

### Mejoras (Features)
- **Gestión de Menús (#1018)**: Se incorporó la administración de campos adicionales (`Additionals`) mediante el uso de _chips input_ en la creación de menús.
  - Soluciona el problema de campos adicionales en la creación de menús (#993).

---

## 📊 POQTROL

### Mejoras (Features)
- **Auditoría de Estadísticas (#1024)**: Implementación de un registro de auditoría (_audit trail_) completo para la generación de estadísticas de glucómetros.
  - Se añadieron registros detallados de las iteraciones del algoritmo.
  - Se implementó la exportación de estos registros de auditoría hacia Firebase Storage.

---

## 🔄 Mi Proceso

### Mejoras (Features)
- **Optimización de Base de Datos (#1020)**: Se agregaron nuevos índices en Firestore para optimizar consultas en las colecciones de `processes`, `processes_history` y `logs`.
