# Release Notes - v4.3.3 🚀
_25 de marzo de 2026_

Esta versión incluye la implementación de un flujo de despliegue manual, un nuevo CRUD para configuraciones de producto en la organización, mejoras en el reporte de participación en POQTROL y limpieza de estilos en Mi Proceso.

---

## 🏗️ zymDev Core (General)

### Mejoras (Features)
- **Flujo de Despliegue Manual (#1037)**: Se introdujo un flujo de despliegue manual y se actualizó el flujo principal para usar Bun en las pruebas, simplificando además los indicadores (*flags*) de despliegue de componentes.
- **CRUD de Configuraciones de Producto (#1022)**:
  - Se actualizó `OrganizationService` para mover la configuración al documento de la organización y segmentarla por entorno de producto.
  - Se implementaron métodos de migración para mover las configuraciones de subcolecciones a mapas anidados por producto.

### Refactorización (Refactor)
- **Atributos Personalizados (#1022)**: Se separó el menú de configuración de los atributos generales para mejorar el diseño de la interfaz de usuario.

### Correcciones (Bugfixes)
- **Pruebas y Seguridad (#1022)**: Se añadieron y actualizaron pruebas unitarias y de integración (*e2e*). Se corrigieron aspectos de seguridad, pruebas unitarias y configuraciones en los componentes de agentes.
- **Migración del Laboratorio (#1022)**: Se limpió código y se corrigió el método `extractAndMigrateConfig` para el nuevo modelo del laboratorio.

---

## 🧪 POQTROL

### Correcciones (Bugfixes)
- **Formato de Fechas y Precisión (#1028)**: Se refinó el formato de fechas y la precisión en las tablas de desempeño para el reporte de participación.
  - Se aplicaron validaciones `isValidNumber` antes de formatear valores numéricos en `calculateZscore`.
  - Se aseguró que el valor `de` se formatee solo cuando es un número válido.

---

## ⚙️ Mi Proceso

### Refactorización (Refactor)
- **Estilos de Columnas (#1036)**: Se eliminaron estilos y efectos de resaltado (*hover*) no utilizados en las columnas de acciones.
