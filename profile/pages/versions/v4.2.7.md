# Release Notes - v4.2.7 🚀
_13 de marzo de 2026_

Esta versión introduce mejoras significativas en la visualización de datos y experiencia de usuario para los reportes de POQTROL.

---

## 📊 POQTROL

### Mejoras (Features)
- **Visualización de Reportes (Preview)**:
  - Se añadió un indicador de carga (_spinner_) al diálogo de vista previa de reportes.
  - Se actualizaron los títulos del diálogo y las gráficas ahora se renderizan condicionalmente, asegurando que solo aparezcan cuando la carga haya finalizado.
  - Se mejoró el formateo y la precisión de los datos presentados en el componente de vista previa.
- **Reportes de Glucómetros**: Se añadieron las respuestas de estabilidad (_stability answers_) a los reportes de glucómetros y se actualizaron las cabeceras correspondientes.

### Otras Mejoras (Refactorings & Chores)
- **Robustez de Código**: Se refactorizó la función `getMapStabilityAnswers` añadiendo definiciones de tipo explícitas y comprobaciones de nulos (_null checks_) para una ejecución más segura y robusta.
