# Release Notes - v4.3.8 🚀
_7 de abril de 2026_

Esta versión incluye mejoras y refactorizaciones importantes en el algoritmo de cálculo de cifras significativas para POQTROL asegurando consistencia y precisión numérica.

---

## 📊 POQTROL

### Refactorizaciones (Refactor)
- **Cifras Significativas**: 
  - Se reescribió la lógica de redondeo para `SignificantFigures`, simplificando su clase al reemplazar el análisis manual de dígitos por un cálculo robusto basado en `Math.round`.
  - Se actualizó el algoritmo para usar el redondeo científico de cifras significativas en lugar del redondeo por posiciones decimales.
  - El comportamiento se alineó con la función `ROUND` de Excel para asegurar la consistencia, incluso en escalas decimales diferentes y valores negativos.
  - Se implementó `toPrecision(15)` en pasos intermedios para mitigar errores de precisión estándares de coma flotante en JavaScript.
  - La propiedad `value_formatted` ahora utiliza `toFixed` para posiciones decimales positivas, mejorando confiabilidad de la representación en forma de cadena.
  - Se agregó una suite completa de pruebas en `significant-figures.spec.ts` para cubrir casos límite, factores de redondeo negativos y verificación de precisión.

### Correcciones (Bugfixes)
- **Cálculo de Cifras Significativas**: Se ajustó el cálculo para evitar el formato decimal incorrecto cuando el redondeo cruza potencias de 10.
